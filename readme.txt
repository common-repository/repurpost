== Installation ==

= For an automatic installation through WordPress: =

1. In your WordPress admin sidebar, go to **Plugins** and then click on **Add New**

2. Search for **Repurpost**

3. Click **Install Now** and activate the plugin

= For a manual installation via FTP: =

1. Go to <a href="https://wordpress.org/plugins/repurpost">https://wordpress.org/plugins/repurpost</a> and click on **Download** to download the Repurpost zipped plugin files

2. Unzip the plugin files

3. In FTP, upload the entire **repurpost** plugin folder you've just unzipped to your `/wp-content/plugins` directory

4. Go back to your WordPress dashboard, click on **Plugins** in the sidebar, find the Repurpost plugin you've just uploaded and click **Activate**

= To upload the plugin through WordPress, instead of FTP: =

1. Go to <a href=https://wordpress.org/plugins/repurpost>https://wordpress.org/plugins/repurpost</a> and click on **Download** to download the Repurpost zipped plugin files (do not unzip this file)

2. Go back to your WordPress admin dashboard, go to **Plugins** and click on **Add New** => **Upload Plugin**

3. Upload the zip plugin files you've just downloaded and click on **Install Now**

4. Once WordPress is done uploading the files, just click on **Activate Plugin**
